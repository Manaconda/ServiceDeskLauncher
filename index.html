Installation.

Download Latest Release (V1.2 Updated 24/09/2015)

It’s an MSI and is designed to be pushed out via Group Policy but it will install normally as well and will prompt if it needs elevation.

How it works.

The Installer adds a shortcut to the All Users Start Menu, launching it at login. It shows a splash screen for a second then puts this Icon:

In the notification area. Left or Right clicking it will being up this panel.

From top to bottom:

• Hostname and IP address of the machine – thought it may come in handy when customers call up with issues.

• A clickable link to the website.

• Log a ticket button – Takes you to service desk. Until I can find a way of making service desk log in users will still have to enter their details (probably best security wise anyway)

• TeamViewer Icon – Launches an instant support Teamviewer session – Useful if the machine doesn’t have Teamviewer installed, plus its bundled in with the app, don’t have to go to a website and download it.

• Close – Closes the panel, not the program.
